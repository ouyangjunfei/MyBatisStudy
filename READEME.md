## 项目环境

- Java : 1.8 (要求)
- IDEA : 2020.3 Ultimate (建议)
- Maven : Bundled (3.6.3)
- MySQL : 5.7.31 (建议使用5.7.x)
- pom.xml的各项依赖没有特别需要注意的，使用最新或者较新版本都可以
- 使用IDEA的Database功能需要下载指定的Driver，这里我使用的是8.x.x的Driver，此处非必须配置，与项目无关

MySQL数据库如果用8.x.x版本涉及到配置参数URL的修改
